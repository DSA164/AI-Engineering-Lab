# Project Taxonomy

## 1. LLM foundations

Goal:

- understand local models;
- sampling;
- inference;
- local API serving.

Projects:

- `2026_project_01--llm_playground`
- `2026_project_02--local_llm_server`

## 2. Workflow automation

Goal:

- ingest data;
- parse APIs / HTML;
- normalize;
- deduplicate;
- score;
- store;
- alert.

Project:

- `2026_project_03--workflow_automation_lab`

Main use case:

- `BD Kids Hunter`

This is not a RAG project.

## 3. RAG systems

Goal:

- retrieve relevant documents or chunks;
- use retrieved context to answer;
- cite sources;
- manage metadata.

Projects:

- `2026_project_04--rag_system`
- `2026_project_05--rag_enterprise`

## 4. Agents

Goal:

- use tools;
- manage state;
- plan actions;
- validate with human-in-the-loop.

Project:

- `2026_project_06--ai_agents_lab`

## 5. Business watch

Goal:

- combine workflow automation, retrieval, memory and agents into useful watch systems.

Project:

- `2026_project_07--business_watch_agent`

## 6. Vision, multimodal, robotics, platform

Projects:

- `2026_project_08--vision_ai_lab`
- `2026_project_09--multimodal_assistant`
- `2026_project_10--robot_ai_system`
- `2026_project_11--ai_platform`
